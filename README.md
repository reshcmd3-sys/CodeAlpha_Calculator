# Web Calculator

A simple and responsive calculator built using HTML, CSS, and JavaScript. This project provides a clean user interface for performing basic arithmetic calculations directly in the browser.

## 🚀 Features

- Perform addition, subtraction, multiplication, and division
- Real-time calculation display
- User-friendly interface
- Responsive design for different screen sizes
- Fast and lightweight

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript 

## 📂 Project Overview

This calculator is designed to help users perform everyday mathematical operations quickly and efficiently. The project demonstrates the fundamentals of front-end web development, including structure, styling, and interactivity.

## 🎯 Objectives

- Build an interactive web application
- Practice DOM manipulation using JavaScript
- Improve HTML and CSS layout skills
- Understand event handling in JavaScript

## ▶️ Getting Started

1. Clone or download this repository.
2. Open the project folder.
3. Run `index.html` in any modern web browser.
